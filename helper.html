FocusPath Tests Page — Complete Guide

Overview

The Tests section lets you create, import, export, and take custom tests to reinforce your learning. You can manage test subjects, build questions, run tests interactively, and track your performance.

1. Sidebar Actions

  • Test Home: Returns you to the main Tests dashboard.
  • New Test: Opens the Test Builder to create a new test from scratch.
  • Import Tests: Allows you to import tests from files or external sources.
  • Export Test: Lets you export your tests for backup or sharing (enabled when a test is selected).
  • Trash All: Deletes all saved tests from local storage.
  • Soft Reset: Hides all tests locally but keeps your stats history.

2. Creating a Test (Test Builder)

  • Access: Click "New Test" in the sidebar.
  • Test Details: Fill in Subject, Subject Component (e.g., History 101), and Description.
    • You can select a subject or create a new one.
    • Manage or remove subjects using the buttons next to the dropdown.
  • Save Test: Click "Save Test" to store your test in local storage.
Building Questions
  • Add Question: Click "+ Question" to add a new question.
  • Bulk Add: Use "Bulk" to add multiple questions at once.
  • Question Editor: Edit question text, type (multiple choice, true/false, etc.), options, and images.
  • Move/Remove: Use the up/down arrows to reorder questions, or delete them.
  • Save Question: Save changes to the current question.
Question Types Supported
  • Multiple Choice: Add options, select the correct answer.
  • True/False: Choose the correct value.
  • Short Answer: Enter the expected answer.
  • Image Support: Attach images to questions for visual reference.

3. Managing Tests

  • Test List: All saved tests appear in the main list. Click a test to view details.
  • Test Info Panel: Shows subject, name, number of questions, creation date, times taken, and time spent.
  • Edit/Delete: Edit or delete tests as needed.

4. Running a Test (Test Runner)

  • Start: Click a test in the list, then choose to "Run" it.
  • Test Runner UI:
    • Title: Displays the test name.
    • Questions: Each question is shown one at a time.
    • Navigation: Use "Back" and "Next" to move between questions.
    • Answer Input: Select or enter your answer (radio buttons for multiple choice/true-false, text for short answer).
    • Images: If a question has an image, it will be displayed.
  • Progress: Your answers are tracked as you move through the test.
  • Completion: At the end, your results are shown, including score and time taken.

5. Importing & Exporting Tests

  • Import: Use the "Import Tests" button to load tests from a file. Supported formats are typically JSON.
  • Export: Select a test and click "Export Test" to save it as a file for sharing or backup.

6. Stats & Achievements

  • Tracking: Your test history, scores, and time spent are tracked and shown in the Statistics section.
  • Achievements: Completing tests and improving scores unlocks achievements.

7. Tips

  • Use the "Bulk" add feature to quickly create many questions.
  • Soft Reset is useful for hiding tests without losing your progress stats.
  • You can manage custom subjects for better organization.
  • Images can be added to questions for visual learning.

FocusPath Flashcards Page — Complete Guide

Overview

The Flashcards section lets you create, import, export, and study custom flashcard sets for any subject. You can build cards, organize sets, run study sessions, and track your progress.

1. Sidebar Actions

  • Flashcard Home: Returns you to the main Flashcards dashboard.
  • New Set: Opens the Card Builder to create a new flashcard set.
  • Import Sets: Allows you to import flashcard sets from files or external sources.
  • Export Set: Lets you export your sets for backup or sharing (enabled when a set is selected).
  • Trash All: Deletes all saved flashcard sets from local storage.
  • Soft Reset: Hides all flashcards locally but keeps your stats history.

2. Creating a Flashcard Set (Card Builder)

  • Access: Click "New Set" in the sidebar.
  • Set Details: Fill in Subject, Subject Component (e.g., Biology Terms 101), and Description.
    • You can select a subject or create a new one.
    • Manage or remove subjects using the buttons next to the dropdown.
  • Save Set: Click "Save Set" to store your flashcard set in local storage.
Building Cards
  • Add Card: Click "+ Card" to add a new card.
  • Bulk Add: Use "Bulk" to add multiple cards at once (format: Front :: Back per line).
  • Card Editor: Edit card front, back, tags, hint, and image.
  • Move/Remove: Use the up/down arrows to reorder cards, or delete them.
  • Save Card: Save changes to the current card.
Card Features Supported
  • Front/Back: Enter the term/question on the front and the answer/definition on the back.
  • Tags: Add tags for organization and filtering.
  • Hint: Add a hint to help during study sessions.
  • Image Support: Attach images to cards for visual learning.

3. Managing Flashcard Sets

  • Set List: All saved sets appear in the main list. Click a set to view details or start a study session.
  • Edit/Delete: Edit or delete sets as needed.

4. Studying Flashcards (Card Runner)

  • Start: Click a set in the list, then choose to "Study" it.
  • Card Runner UI:
    • Meta Info: Shows subject and card number.
    • Card Stage: Displays the current card.
    • Flip: Click the card to flip between front and back.
    • Navigation: Use "Previous" and "Next" to move between cards.
    • Hint: Click "Hint" to show the card's hint (if available).
    • Finish: Click "Finish" to end the session and log your study.
    • Return: Click "Return to Set" to go back to the set list.
  • Progress: Your study sessions are tracked, including cards seen and time spent.

5. Importing & Exporting Sets

  • Import: Use the "Import Sets" button to load sets from a file. Supported formats are typically JSON.
  • Export: Select a set and click "Export Set" to save it as a file for sharing or backup.

6. Stats & Achievements

  • Tracking: Your flashcard study history, cards seen, and time spent are tracked and shown in the Statistics section.
  • Achievements: Completing study sessions and reviewing cards unlocks achievements.

7. Tips

  • Use the "Bulk" add feature to quickly create many cards.
  • Soft Reset is useful for hiding sets without losing your progress stats.
  • You can manage custom subjects for better organization.
  • Images and hints can be added to cards for enhanced learning.

Dictionary Help Guide

Welcome to the FocusPath Dictionary! This tool helps you build, organize, and search key terms and definitions for any subject. Your dictionary powers future tests and games, making it a central resource for your learning.

Main Features:

  1. Term List Editor (Left Panel)
    • Subject Selector: Choose a broad subject (Art, Math, Science, etc.) for your term list.
    • Component Subject: Optionally specify a class or component (e.g., "Art 201").
    • Term Staging List: Add terms and definitions here before saving.
    • Save List: Stores your current term list for future use.
    • View List: Opens the selected list for review and editing.
    • Delete List: Removes the selected list permanently.
    • Trash All: Deletes all saved term lists and their terms.
  2. Dictionary Search & Add (Center Panel)
    • Search Bar: Type a word or phrase and click "Lookup" or press Enter.
    • Lookup: Searches for definitions using an online dictionary API. Results show the word, meanings, and definitions.
    • Add: After searching, click "Add" to stage the word and its definition for your term list.
    • Results: Definitions are displayed with part of speech and example usage (if available).
  3. Term List Display
    • Terms List: Shows all terms and definitions in your current list. Scroll to review or edit.
    • List Detail View: Opens when you view a saved list, allowing for further editing or exporting.
  4. Statistics Tracking
    • The app tracks your usage invisibly (searches, lists created/updated/deleted, exports to flashcards, time spent searching) to help you monitor your study habits.

How to Use:

  • Build a Term List: Select a subject, add terms and definitions, then save. You can create multiple lists for different subjects or classes.
  • Search for Definitions: Use the search bar to look up words. Add useful definitions directly to your term list.
  • Review & Edit: View saved lists, edit terms, or delete lists as needed.
  • Export to Flashcards: Use your dictionary lists to generate flashcards for study and review (see Flashcards Help for details).

Tips:

  • Use concise, clear definitions for best results in tests and games.
  • Organize terms by subject and class for easy retrieval.
  • Regularly review and update your lists to keep them relevant.
  • Use the "Trash All" button with caution—it deletes all your saved lists!

Privacy:

  • All data is stored locally in your browser. No terms or definitions are sent to external servers except for dictionary lookups.

Troubleshooting:

  • If a word isn’t found, try a simpler form or check your spelling.
  • If buttons are disabled, ensure you have selected or staged a list/term.

Games Help Guide

FocusPath offers experimental learning games to reinforce your study concepts. These games use your saved flashcards and dictionary terms to create interactive challenges. All progress is tracked locally for privacy.

Main Games Page

  • Sidebar & Navigation: Access all FocusPath tools. Motivational Marty appears in the sidebar for encouragement.
  • Game Cards: The main grid displays available games. Click a card to launch a game.
  • Session Tracking: Games only track your progress for the current session (not across devices).

Game Types & Functions

  • Flash Card Sprint
    • Purpose: Rapidly review flashcards from your saved sets.
    • Settings: Choose total time (seconds) and which subjects to include.
    • Gameplay: Cards are shown one by one; answer as quickly as possible.
    • Progress: Tracks accuracy and speed for the session.
  • Concept Typer
    • Purpose: Practice recalling key terms from their definitions.
    • Gameplay: A definition is shown. Type the correct term and press Enter. Immediate feedback: "Correct!" or the correct answer if missed. Click "Next" to proceed to the next concept. Stats are shown: accuracy, average time per answer. At the end, see your final accuracy and average time.
    • Concepts: Uses a built-in set of science terms and definitions (can be expanded in code).
  • Quick Math Drill
    • Purpose: Test and improve your mental math speed and accuracy.
    • Settings: Set per-question time (seconds) and select which operations (+, -, ×, ÷) to include.
    • Gameplay: Solve randomly generated arithmetic problems before the timer runs out. One mistake or timeout ends the game.
    • Progress: Tracks score, accuracy, and time per question. Review all attempts at the end, with correct/incorrect answers and time taken.
    • Tips: Use settings to focus on specific operations or increase difficulty. Play repeatedly to improve speed and accuracy.
  • Other Games
    • More games are planned (spaced repetition sprint, formula builder, logic rush).

How to Use

  • Open the Games Page: Click "Games" in the sidebar.
  • Select a Game: Click a game card to launch.
  • Adjust Settings (if available): For games like Flash Card Sprint, set your preferred time and subjects.
  • Play: Follow on-screen instructions. Most games provide instant feedback and track your stats.
  • Review Results: At the end of each game, review your accuracy and speed.

Tips

  • Use your own flashcard and dictionary sets for personalized practice.
  • Try Concept Typer to improve recall of definitions and key terms.
  • Play regularly to reinforce learning and track your improvement.

Privacy

  • All game progress and stats are stored locally in your browser.
  • No data is sent to external servers.

Troubleshooting

  • If no flashcard sets are available, create some in the Flashcards section first.
  • If a game doesn't load, refresh the page or check your browser's local storage settings.

Statistics & Achievements Help Guide

The Statistics page provides a detailed overview of your study progress and performance across all FocusPath tools. It helps you visualize your learning journey, identify strengths and areas for improvement, and stay motivated.

Main Functions:

  • Stats Home: View a summary of your overall study activity.
  • Test Stats: See data on tests created, attempts, scores, and trends over time. Charts show your average scores, subject performance, and study streaks.
  • Flashcard Stats: Track flashcard sets created, cards studied, session counts, and subject breakdowns.
  • Dictionary Stats: Monitor term lists, searches, and usage patterns.
  • Trophy Room: View all achievements and badges earned for study milestones and skill mastery.
  • Hard Reset: Erase all study data except achievements (use with caution).

Why is it Useful?

  • Progress Tracking: See how much you’ve studied, your test scores, and flashcard mastery.
  • Motivation: Visual charts and streaks encourage consistent study habits.
  • Goal Setting: Achievements and badges help you set and reach learning goals.
  • Self-Assessment: Identify subjects or skills that need more attention.
  • Privacy: All stats are stored locally in your browser.

Achievements System

  • Automatic Tracking: Achievements are unlocked as you reach milestones (e.g., creating tests, scoring high, maintaining streaks, studying multiple subjects).
  • Badges & Trophies: Earn badges for activities like “First Test Created”, “Math Mastery”, “Flashcard Marathon”, and more.
  • Notifications: Achievement notifications appear outside active study/test pages, so they don’t interrupt your focus.
  • Trophy Room: View all earned achievements and see which ones you’re close to unlocking.

Why Achievements Matter

  • Motivation: Celebrate your progress and keep pushing for new goals.
  • Skill Mastery: Badges highlight your strengths and encourage balanced study.
  • Streaks & Consistency: Streak achievements reward regular study habits.
  • Personalized Goals: Achievements adapt to your study style and subjects.

Tips

  • Visit the Statistics page regularly to review your progress.
  • Use achievements as motivation to try new subjects and maintain study streaks.
  • All data is private and stored locally—your stats and achievements are yours alone.

Event Planner Help Guide

The Event Planner in FocusPath helps you organize your study schedule, track important deadlines, and stay motivated. It’s designed to make planning and managing your academic life simple and effective.

Main Features

  • Calendar View: See your month at a glance. Navigate between months using the arrow buttons.
  • Upcoming Events List: Displays all your upcoming events, deadlines, and study sessions in a scrollable list for quick reference.
  • Create Event Form: Add new events with details like date, time, type, and description. Double-click a date on the calendar to autofill the date in the form.
    • Date & Time: Specify the exact date and time, or mark events as “All Day.”
    • Event Type: Choose from Test/Exam, Study Time, Group Activity, Practice, and more.
    • Description: Add notes or details to help you remember what the event is about.
  • Event Autofill: Double-clicking a date on the calendar automatically fills the event form with that date.
  • Event Management: Easily add, view, and track events. Events are displayed both on the calendar and in the upcoming list.

Why is it Useful?

  • Organization: Keep track of all your study-related events, deadlines, and goals in one place.
  • Time Management: Plan your study sessions, exams, and group activities to avoid last-minute stress.
  • Motivation: Seeing your schedule and progress helps you stay focused and motivated.
  • Reminders: Never miss an important test, study session, or group activity.
  • Customization: Add detailed notes and choose event types to fit your personal study style.
  • Privacy: All events are stored locally in your browser—no data is sent to external servers.

Tips

  • Use the calendar to plan your study week and set realistic goals.
  • Add all your exams, assignments, and study sessions as events for better tracking.
  • Review the Upcoming Events list regularly to stay on top of your schedule.
  • Double-click dates on the calendar for quick event creation.
